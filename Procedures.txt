To create a new API request:

1.  Create API Request in Domain/(Area)/							PutBankAPIRequest
2.  Create API Request validator									PutBankAPIRequestValidator
3.  Create Query/Command Request									PutBankCommandRequest
4.  Map API Request to Query/Command				BLL.Mappings	PutBankAPIRequest, PutBankCommandRequest
5.  Create Query/Command Response									PutBankCommandResponse
6.  Create API Response												PutBankAPIResponse
7.  Map Query/Command Response to API Response						PutBankCommandResponse, PutBankAPIResponse
8.  Create the Handler												PutBankCommandHandler
9.  Put the action into the appropriate Service						_bankService.Add()
10. Put the action into the appropriate Repository						_bankRepository.Add()
